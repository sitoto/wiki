Mongoid

  1. MongoDB/Mongoid字段为 Boolean 属性的声明，:tpye => Boolean ,生成的数据为 1/0,实际应该为 true/false; 需要声明为：
    • field :locked, :type => Mongoid::Boolean, :default => false
  2. 'user' belongs_to :last_agree_user, :class_name => 'User'
  3. index :status => 1 & index :status => -1 #1/-1表示创建索引使用正序/倒序
  4. scope :draft, -> { where(status: 0) }
