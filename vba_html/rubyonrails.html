Mongoid

    • field :locked, :type => Mongoid::Boolean, :default => false
  2. 'user' belongs_to :last_agree_user, :class_name => 'User'
  3. index :status => 1 & index :status => -1 #1/-1表示创建索引使用正序/倒序
  4. scope :draft, -> { where(status: 0) }

数据分组

@students=Student.all
@student_groups=@students.group_by{|s| s.gender}

那么现在得到的@student_groups就有两组,一组是male,一组是female.在对其进行循环的时候,使用hash循环的方式:

<% @student_groups.each do |gender,students| %>
<%= gender %>
<ul> <% students.each do |student| %> <li><%= student.name%></li> <% end %></ul>
<% end %>

结果如下: female

lucy jessi

male

jack jim mike
